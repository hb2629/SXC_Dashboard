# SXC Dashboard

This is a React dashboard displaying financial models and charts.

## Live Demo

View the interactive dashboard at: [https://hb2629.github.io/SXC_Dashboard](https://hb2629.github.io/SXC_Dashboard)

## Description

The dashboard includes margin models, regressions, and two-tier GP calculations, visualized with charts using Recharts.

## Installation

To run locally:

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
